Visualizing the sampling distribution of the mean

Ven 12 mars 2021
Category: Statistics

A lot of science and engineering today uses inferential statistics. Which is all about getting information about a population from a sample. The mean of the sample is commonly used to infer the mean of the population. It happens to have a nice property that makes it easier to handle:

The sampling distribution of the mean is normally distributed around the population mean (well mostly, because of course "terms and conditions apply").

And this is something we can visualize.

In [1]:
import pandas as pd
import numpy as np
import matplotlib.pyplot as plt
import matplotlib.ticker as mticker

Let us start by creating a population that is not normally distributed. I will work with functions to be able to try this out easily with many distributions.

I will start by simulating a population for which I measured a variable Value, that happens to follow a Lomax (or Pareto II) distribution. This distribution has many applications in business and economics.

  • Distribution: https://en.wikipedia.org/wiki/Lomax_distribution
  • Implementation: https://numpy.org/doc/stable/reference/random/generated/numpy.random.pareto.html

Population Lomax distribution¶

In [2]:
def generate_rand_lomax(n):
    a, m = 3., 2.  # shape and mode
    s = (np.random.pareto(a, n) + 1) * m  # np.random.pareto makes a Lomax
    return s

Let us create one population with 100,000 observations...

population_lomax = generate_rand_lomax(n=100000)

... and see what it looks like.

In [4]:
def plot_population_distribution(data):
    # plotting the distribution and its mean
    fig, ax = plt.subplots(figsize=(10, 6))

    # the weights are used to display percentages of samples
    n = len(data)
    weights_pc = np.array([1.]*n) / n

    # adding the histogram
    ax.hist(data, bins=100, weights=weights_pc, color="tab:blue")
    
    ax.axvline(
        data.mean(), 
        color="r", linestyle="solid", linewidth=1, 
        label="Population mean = {:.3f}".format(data.mean())
    )
    
    # formatting
    y_ticks_loc = ax.get_yticks().tolist()
    ax.yaxis.set_major_locator(mticker.FixedLocator(y_ticks_loc))
    ax.set_yticklabels(["{:,.1%}".format(y) for y in y_ticks_loc])
    
    ax.set_xlim(min(data.min(), 0))

    ax.set_ylabel("Percentage of observations", fontsize=12)
    ax.set_xlabel("Value", fontsize=12)
    ax.legend(loc="upper right")

    return fig

Preparing the sampling¶

Now that we have a nice population, we can make a function that will:

  • take a sample of size sample_size
  • compute the means and save it in means
  • repeat a number of times equal to number_of_samples

That will give us a list of means.

In [6]:
def compute_sample_means(
    population_data, sample_size, number_of_samples, means=None):

    if means is None:
        means = list()
    for i in range(0, number_of_samples):
        sample = np.random.choice(population_data, sample_size, replace=False)
        sample_mean = np.mean(sample)
        means.append(sample_mean)
    
    return means

We will then use a function to plot that list on a histogram, together with the population mean.

To adapt the layout, I will keep some parameters flexible (figsize, xlim, ylim).

In [7]:
def plot_samples_distribution(
    means, pop_mean, bins=100, figsize=(10, 6), xlim=None, ylim=None):

    fig, ax = plt.subplots(figsize=figsize)
    ax.hist(means, bins)
    ax.axvline(
        pop_mean, color="r", linestyle="solid", linewidth=1,
        label="Population mean = {:.3f}".format(pop_mean)
    )
    ax.axvline(
        np.mean(means), color="g", linestyle="dashed", linewidth=1,
        label="Mean of sample means = {:.3f}".format(np.mean(means))
    )
    
    # formatting 
    if xlim is not None:
        ax.set_xlim(xlim)
    ax.set_xlabel("Average value of samples", fontsize=12)
    if ylim is not None:
        ax.set_ylim(ylim)
    ax.set_ylabel("Number of samples", fontsize=12)
    ax.legend(loc="upper right")

    return fig

Showing step-by-step the normal distribution of the means¶

Our population has 100,000 observations, so we will take samples of 2000 observations (luckily it happens to be more than 30 and less than 10% of the population).

The green line is the mean of all the sample means displayed. The red line is the population mean. Both already are very close.

In [10]:

    lomax_sample_means[:1000], np.mean(population_lomax), 
    ylim=(0, 50), xlim=(2.85, 3.15)
)

With 1,000 sample means, the bell curve starts to appear clearly, but still a lot of irregularities. Both the population mean and samples means are now matching: the distribution of the sample means is centered around the population mean. The population mean is equal to the mean of the sample means.

In [11]:
fig22 = plot_samples_distribution(
    lomax_sample_means, np.mean(population_lomax), 
    ylim=(0, 500), xlim=(2.85, 3.15)
)

With 10,000 sample means, the curve is just a plain bell-shaped one: the sampling distribution really looks normally distributed.

With more samples added the curve gets only smoother.

def generate_rand_triangular(n):
    l, m, r = -1, 0, 5  # left limit, mode, right limit
    return np.random.triangular(l, m, r, n)

Toward normality¶

We could try many more distributions, and the results would be the same, all thanks to the Central Limit Theorem (https://en.wikipedia.org/wiki/Central_limit_theorem). This is key to many usages of statistics, including hypothesis testings.

Some points here need to be underlined regarding this exercise (you know, the "terms and conditions apply"). For the Central Limit Theorem to work:

  • The variables should be independent from each other: sampling should always be random, and in the case of a sampling without replacement, the sample should not be too big (let us keep <10%).
  • It is a normal distribution if n -> infinite (let us say minimum 30 observations, as we saw in the example above, that worked fine).
  • Population distribution should have a finite variance: that means that it does not work with some specific population distributions (those are not very common).

What happens when the condition are not met?¶

Those conditions directly impact the quality of modelling sample means as a normal distribution. The further from those conditions, the further away the sample means will be from an actual normal distribution. The condition n -> infinite forces us to get a large sample, like we did. Let us see the impact of a small sample.

In [17]:
lomax_10_sample_means = compute_sample_means(
    population_lomax, sample_size=10, number_of_samples=10000)
In [18]:
fig50 = plot_samples_distribution(
    lomax_10_sample_means, np.mean(population_lomax))

Clearly, this is not normally distributed anymore. The closer the population distribution is from a normal distribution, the closer the sampling distribution of means derived from it will be. We can see that with the triangular distribution for which samples of size 10 already look normally distributed. If the underlying variable in already normally distributed, it is possible to use small samples.
