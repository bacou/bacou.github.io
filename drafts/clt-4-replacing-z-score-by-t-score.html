Replacing z-scores by t-scores (CLT 4)

07/09/2021
Catégorie : statistiques

In this article we saw how to build a confidence interval around a sample mean to make an estimation of the population mean. We noticed that, for small samples, our confidence intervals do not seem to work anymore. For that reason the z-score we used is replaced by a t-score.

What was the z-score again? The z-score, z-value or standard score is the number of standard deviation between an observation and the mean, in the case of a normal distribution. For example, we can use it the say that 95.45% of the values lay within the 2 first standard deviations around the mean. In that case the "2" is a z-score. In the same context, a z-score of "1" gives 68.27% of the observations, a z-score of "3" gives 99.73%.

def plot_population_distribution(data):
    # plotting the distribution and its mean
    fig, ax = plt.subplots(figsize=(10, 6))

    # the weights are used to display percentages of samples
    n = len(data)
    weights_pc = np.array([1.]*n) / n

    # adding the histogram

    ax.axvline(

    )
    
    # formatting
    y_ticks_loc = ax.get_yticks().tolist()

    ax.set_ylabel("Percentage of observations", fontsize=12)
    ax.set_xlabel("Value", fontsize=12)
    ax.legend(loc="upper right")

    return fig

We have a nice normal population. Now we will collect multiple samples, and save their means and standard deviations. We will start with large samples before going for our problematic small samples.

Note: In order to use the sample standard deviation as an estimator of the population standard deviation, we need it to be unbiased. This require to divide by n−1 rather than n. In the code implementation of the formula, we use the parameter ddof for delta degrees of freedom set to 1. More on the n-1 here.

In [6]:
def compute_sample_means(population, sample_n, n_samples):

    sampling_rows = []
    for i in range(0, n_samples):
        sample = np.random.choice(population, sample_n, replace=False)
        sample_dict = {"mean": np.mean(sample), "std": np.std(sample, ddof=1)}
        sampling_rows = [*sampling_rows, sample_dict]

    return pd.DataFrame(sampling_rows)

The standard normal distribution¶

Any normal distribution can be transformed into a standard normal distribution using a simple transformation. To each observation of the distribution, we substract the mean of the distribution and divide the result by the standard deviation of the distribution. This is called standardization and the resulting distribution is a normal distribution with a mean of 0 and a standard deviation of 1.

In our case we can create the standard normal distribution of the sampling distribution of means by substracting the mean of the population (equal to the mean of the sample means) and divide the result by the standard error of the mean (equal to the population standard deviation divided by the square root of the sample size).

def plot_distribution_with_range(
    data, selected_center, selected_range, 
    bins=100, figsize=(10, 6), xlim=None, ylim=None):

    fig, ax = plt.subplots(figsize=figsize)

    # distribution histogram
    bins_n, bins_val, rectangles = ax.hist(data, bins)

    # distribution mean
    ax.axvline(
        np.mean(data), color="g", linestyle="dashed", linewidth=1,
        label="Mean of sample means = {:.3f}".format(np.mean(data)))
    
    # finding the bin of our selected center
    central_bin = bins_val[bins_val < selected_center].max()
    central_index = np.where(bins_val == central_bin)[0][0]

    # finding the bottom bin of our range
    lower_mean = selected_center - selected_range
    lower_bin = bins_val[bins_val < (lower_mean)].max()
    lower_index = np.where(bins_val == lower_bin)[0][0]

    # finding the top bin of our range
    upper_mean = selected_center + selected_range    
    upper_bin = bins_val[bins_val < (upper_mean)].max()
    upper_index = np.where(bins_val == upper_bin)[0][0]
    
    # coloring
    for i in range(lower_index, upper_index):
        rectangles[i].set_color("navajowhite")
    
    # label for the navajowhite bars
    n_samples = len(data)
    n_true = len([s for s in data if (s <= upper_mean) and (s >= lower_mean)])
    ax.add_patch(patches.Rectangle(
        (0, 0), 0, 0,
        facecolor = 'navajowhite',
        label="Pc of samples between -{} and {} = {:.2%}".format(
            selected_range, selected_range, (n_true / n_samples))
     ))
    
    # formatting 
    if xlim is not None:
        ax.set_xlim(xlim)
    ax.set_xlabel("Average value of samples", fontsize=12)
    if ylim is not None:
        ax.set_ylim(ylim)
    ax.set_ylabel("Number of samples", fontsize=12)
    ax.legend(loc="upper right")

    return fig

Standardization when SEM is estimated¶

When a single sample is available, we use this sample standard deviation to compute the standard error of the mean, equal then to the sample standard deviation divided by the square root of the sample size.

To be consistent, we should change our way of standardizing and see if it has an impact. Let us try this out.

In [16]:
large_sample_means["t-score"] = (
    (large_sample_means["mean"] - normal_population_mean) / (large_sample_means["std"] / (large_sample_size**(1/2)))
)
In [17]:
fig_adj_stdz_large_sample_means = plot_sample_means_distribution(
    large_sample_means["t-score"],

)

We can see that this new distribution is not exaclty the same as the one above, but it still looks normal. We can compare both: the standard normal distribution (the one of the z-scores) will be in orange and the adjusted standard normal distribution (the last one with an estimated SEM) will be in blue.

def plot_distributions_comparison(
    means, pop_mean, bins=100, figsize=(10, 6), xlim=None, ylim=None):

    means_0, means_1 = means
    fig, ax = plt.subplots(figsize=figsize)

        [means_0, means_1], bins, 
        label=([means_0.name, means_1.name])
    )
    ax.axvline(
        np.mean(means), color="g", linestyle="dashed", linewidth=1,
        label="Mean of sample means = {:.3f}".format(np.mean(means))
    )
    
    # formatting 
    if xlim is not None:
        ax.set_xlim(xlim)
    ax.set_xlabel("Average value of samples", fontsize=12)
    if ylim is not None:
        ax.set_ylim(ylim)
    ax.set_ylabel("Number of samples", fontsize=12)
    ax.legend(loc="upper right")

    return fig

This is far from the 95.45% expected with a z-score. We need to adjust the multiplicator of our confidence internval to that new distribution. The replacement for the z-score is the t-statistic.

To reach the same level of confidence for that sample size, we will need to go to 4.527.

The t-statistic depends on this distribution, called Student Distribution. We actually have many Student distributions, as there is a different one for every sample size. The bigger the sample, the closer the distribution is to the normal distribution, as we saw with our large sample example.

Replacing the z-score with our t-statistic gives us a reliable method to build a confidence interval in case of small samples.
